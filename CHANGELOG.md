# Changelog

All notable changes to the DevStack Extension will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [0.1.0] - 2025-10-31

### Added
- Initial release of DevStack Extension
- `hi` command - Displays a friendly greeting and extension info
- Cross-platform build scripts (build.sh and build.ps1)
- Support for Windows, Linux, and macOS (AMD64 and ARM64)
- Basic extension structure with cobra CLI framework

### Features
- Custom command capability
- Developer productivity focused commands
- Clean and simple CLI interface
